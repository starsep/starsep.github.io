I am a big fan of open source software. I try to use it whenever possible.

Often it is maintained by a single developer in their free time.
To help them I’d like you to:

  1. Try the software
  2. Submit bug reports
  3. Contribute to the development if you have such skills
  4. Donate if you can

To help spread the word I want to share what works for me. My priorities are:

  • usability
  • simplicity
  • control over software and my data
  • privacy
